== Description ==
Filtry is a powerful and user-friendly plugin that supercharges your WooCommerce store with lightning-fast product filters. Enhance the shopping experience, help your customers find what they need quickly, and boost your sales with this essential WooCommerce add-on.

**Key Features:**
* Lightning-fast AJAX-based filtering.
* Seamless integration with WooCommerce.
* Customizable filter options (price, categories, tags, attributes).
* Mobile-friendly and responsive design.
* Easy setup and configuration with no coding required.

Filtry is designed to make shopping easier and faster, improving conversion rates and customer satisfaction.

== Manual Installation ==
1. Upload the `filtry` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Filtry Settings to configure your filters.
4. Add the filters widget to your shop sidebar.

== Frequently Asked Questions ==

= Does Filtry work with any WooCommerce theme? =
Yes, Filtry is designed to work seamlessly with most WooCommerce-compatible themes.

= Can I customize the filter appearance? =
Yes, you can customize the filter options and appearance through the settings panel.

= Does Filtry use AJAX for faster filtering? =
Absolutely! Filtry uses AJAX to provide instant results without page reloads, enhancing the user experience.
